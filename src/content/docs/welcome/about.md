luxe is an easy to use, cross platform game engine with a powerful and flexible foundation.
   
It has been lovingly crafted to offer tools that work with you, whether you're a first time game developer, or have a long history in games.

## Getting started

- **To get started, you'll want to [install](/welcome/get/) luxe.**
    - A step by step guide to using the luxe launcher.
- **Read the [Luxe 101 guide](/getting-started/luxe-101/).**
    - Learn about the big picture of how luxe is designed!
- **Follow [the getting started series](/getting-started/overview/).**
    - Creating a project, drawing stuff on screen, building your game and more.
- **The focused [tutorials](/tutorials/overview/) are next.**
    - These teach you the basics of interacting with the engine.

## And then...

- **Dig deeper using [the manual](/manual/overview/).**
    - The manual includes in depth information about the engine and systems.
- **Scripting reference is found in the [API reference](https://api.luxeengine.com).**
    - This is the place to search for API usage and documentation.
